 **.zsh** - my zsh configuration file


## how to use it
```shell
brew install zsh
sudo chsh -s $(which zsh) $(whoami)
cd ~ && git clone https://github.com:yanyingwang/.zsh && echo '. $HOME/.zsh/zshrc' >> .zshrc
```
Open a new terminal and wait for a second, everything will be setted.

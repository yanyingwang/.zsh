# how to use it
```shell
brew install zsh
sudo chsh -s $(which zsh) $(whoami)
git clone git@gitlab.com:yanyingwang/.zsh.git $HOME && \
echo '. $HOME/.zsh/zshrc' >> $HOME/.zshrc
```
Open a new terminal and wait for a second, everything will be setted.
